GIS Projects and Codes -

Programming For Geographical Information Analysis

  • Assessment 1 - Agent Based Model (ABM)

    • A simple ABM implemented in Python, displaying 10 agents moving randomly and interacting with each other and resources within their environment by eating, collecting and storing resources and sharing resources with neighbours. The implementation uses Python list, numpy arrays for storing and manipulating data and matplotlib for plotting.

  • Assessment 2 - Black Death

    • Implemented in Jupyter notebook to show the scale of deaths in the Black Death of 1665. The plague was attributed to fleas from black rats infecting people. The model maps the population densities of 16 parishes in London and the rats caught. It outputs the average weekly death and total weekly death and also a map for the average weekly death.

Geocomputation -

  • NetLogo - Assessment 2

    • Implemented in Netlogo this is an ABM modelling the flow of a crowd from a building during an emergency and it shows how the influence the following have on crowd exiting the building: the design of the building - the placement of pillar near the exit and the width of exit, the age and mobility of people in the crowd.

Spatial Analysis -

  • ArcGIS - Assessment 2

    • An exploration of patterns in crime data for the wards in Leeds, UK, for the period 2000-03 using ArcGIS and spatial analytical and mapping techniques such as Thematic point based maps, Choropleth maps, Kernel Density, Getis Ord Gi* and Geographical Analysis Machine (GAM).

Using GIS -

  • Assessment1

    • A project to investigate if there is a link between proximity to motorways and the rate of Limiting Long Term Illness (LLTI) in a selected county of England, Buckinghamshire. Data used from the 2001 Census. Tools used ArcGIS and Excel.

  • Assessment 2

    • A project to use ArcGIS Spatial Analyst to create a land capability map for agricultural planning for an area in Scotland using the British Land Capability Assessment (LCA) and Multi-Criteria Evaluation (MCE) and compare and critique both methodologies.
